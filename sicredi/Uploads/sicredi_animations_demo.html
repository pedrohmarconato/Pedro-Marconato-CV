Animações do Logo Sicredi

Demonstração de animações dotLottie para o logo da Sicredi, incluindo movimento do símbolo e rotação de cores.

Movimento do Símbolo

Esta animação mostra o símbolo do catavento se separando do texto do logo Sicredi, movendo-se para a esquerda e girando em 360 graus. O movimento fluido representa dinamismo e inovação, valores essenciais da marca Sicredi.

Rotação de Cores

Esta animação apresenta o logo da Sicredi com as cores girando como uma engrenagem. As cores transitam entre diferentes tonalidades de verde e voltam para a cor original verde (#009A49), simbolizando a versatilidade e a natureza cooperativa da marca.

Como Implementar Estas Animações

1. Incluir a biblioteca dotLottie Player

Adicione o script da biblioteca dotLottie Player no cabeçalho do seu HTML:

<script src="https://unpkg.com/@dotlottie/player-component@latest/dist/dotlottie-player.js"></script>

2. Adicionar o Player ao HTML

Insira o elemento dotlottie-player no local onde deseja exibir a animação:

<!-- Para a animação de movimento -->
<dotlottie-player
  src="caminho/para/sicredi_movement.lottie"
  background="transparent"
  speed="1"
  style="width: 300px; height: 300px;"
  loop
  autoplay
  id="movement-player">
</dotlottie-player>

3. Controlar a Animação via JavaScript

Você pode controlar a animação usando JavaScript:

// Reproduzir a animação
document.getElementById('movement-player').play();

// Pausar a animação
document.getElementById('movement-player').pause();

// Parar/Reiniciar a animação
document.getElementById('movement-player').stop();

// Definir a velocidade da animação
document.getElementById('movement-player').setSpeed(1.5); // 1.5x mais rápido

4. Opções de Personalização

O player dotLottie oferece várias opções de personalização:

  • loop: Define se a animação deve repetir (true/false)
  • autoplay: Define se a animação deve iniciar automaticamente
  • speed: Controla a velocidade da animação (1 = normal)
  • background: Define a cor de fundo (ex: "transparent", "#ffffff")
  • controls: Exibe controles nativos do player

5. Responsividade

Para garantir que a animação seja responsiva, você pode usar CSS:

dotlottie-player {
  width: 100%;
  max-width: 500px;
  height: auto;
}
